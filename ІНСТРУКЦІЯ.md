# FaceRecog2 — Інструкція з управління базою облич

## Веб-інтерфейс

Відкрий в браузері: **http://192.168.1.26:8080**

Доступно з будь-якого пристрою в домашній мережі (ПК, телефон).

---

## Як додати нові фото особи

### Спосіб 1 — через веб-інтерфейс (рекомендовано)

1. Відкрий **http://192.168.1.26:8080**
2. Знайди блок **"Додати фото особи"**
3. У випадаючому списку **"Вибрати особу"** — обери ім'я (або "+ Нова особа...")
4. Натисни **"📁 Обрати"** — вибери 1 або кілька фото з телефону/ПК
5. Натисни **"Додати фото"**
6. Система автоматично витягне ембедінг і оновить базу — нічого більше не потрібно

> Якщо написало "✗ Жодне фото не прийнято — обличчя не виявлено" — на фото обличчя не знайдено.
> Перевір: обличчя чітке, фронтальне, добре освітлене.

### Спосіб 2 — через розділ "Невідомі обличчя"

Коли система бачить незнайоме обличчя — зберігає знімок у розділ **"Невідомі обличчя"**.

1. Відкрий **http://192.168.1.26:8080** → розділ **"Невідомі обличчя"** → натисни **"Оновити"**
2. Знайди фото потрібної людини
3. У випадаючому списку поруч — обери ім'я особи (або введи нове)
4. Натисни **"→"** — фото буде додано до бази автоматично

## Які фото давати для кращого розпізнавання

Для кожної людини потрібно **мінімум 5–10 фото**, різних:

| Параметр | Що потрібно |
|---|---|
| Ракурс | Прямо, трохи вліво, вправо, злегка зверху |
| Освітлення | Вдень природне, ввечері штучне, різні кімнати |
| Відстань | Близько (обличчя велике) + далі (як в камери) |
| Вираз | Нейтральний, легка посмішка |

**Важливо для cam_65**: камера дивиться зверху вниз. Зроби кілька фото де голова трохи нахилена вниз — як коли дивишся в підлогу/телефон.

---

## Перегляд і видалення фото особи

1. Відкрий **http://192.168.1.26:8080** → блок **"Особи в базі"**
2. Натисни на картку особи — відкриється список всіх її фото
3. Натисни на фото — відкриється повний розмір
4. Натисни **червону кнопку ✕** — фото видалиться, база оновиться автоматично

Щоб видалити особу повністю — натисни **✕** в куті картки особи.

---

## Перегляд знімків розпізнавання

Розділ **"Останні знімки"** — всі випадки коли система впізнала людину.

- Фільтр по імені — випадаючий список зверху
- **"▶ Авто"** — автооновлення кожні 5 секунд
- Натисни на знімок — відкриється повний розмір

## Налаштування порогів (config.yml на сервері)

Файл: `/opt/facerecog2/app/config.yml`

```yaml
recognition:
  similarity_threshold: 0.50   # нижче цього — "unknown" (зменшити якщо пропускає знайомих)
  unknown_threshold: 0.38      # нижче цього — взагалі не зберігати як unknown
  cooldown_sec: 30             # пауза між повторними сповіщеннями (секунди)
  det_score_min: 0.3           # мінімальна впевненість детектора обличчя

cameras:
  cam_65:
    min_face_w: 50             # мінімальна ширина обличчя в пікселях (зменшити якщо пропускає)
```

Після зміни config.yml — перезапустити сервіс:
```
systemctl restart facerecog2
```

## Якщо система не впізнає людину

**Крок 1** — перевір скільки фото в базі:
- Відкрий http://192.168.1.26:8080 → "Особи в базі" → натисни на особу

**Крок 2** — перевір невідомі:
- Якщо людина з'являється в "Невідомі обличчя" — система бачить обличчя, але не впізнає
- Додай більше різноманітних фото цієї людини

**Крок 3** — якщо немає навіть в невідомих:
- Людина стоїть занадто далеко від камери (обличчя < 50px шириною)
- Обличчя сильно повернуте в сторону або вниз
- Погане освітлення

## Логи

Розділ **"Логи сервісу"** в веб-інтерфейсі — останні 80 рядків.

Або на сервері:
```
journalctl -u facerecog2 -f
```
